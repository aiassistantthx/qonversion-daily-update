## Как использовать

### Для окупаемости (D180 = 100%)
1. **D7 < 20** — скорее всего кампания не окупится, рассмотреть остановку
2. **D30 < 35** — риск не достичь 100% на D180
3. **D90 > 77** — кампания на пути к успешной окупаемости

### Для маржинальности (D180 = 120%)
1. **D7 < 24** — скорее всего кампания не достигнет 120%
2. **D30 < 46** — риск не достичь 120% на D180
3. **D90 > 92** — кампания на пути к 120% ROAS

## Запросы для анализа

```sql
-- Когортный ROAS по кампании
SELECT * FROM cohort_roas_data WHERE campaign = 'YOUR_CAMPAIGN' ORDER BY cohort_month;

-- Топ кампании по D180
SELECT campaign, SUM(spend),
  ROUND((SUM(rev_d180)/NULLIF(SUM(spend),0)*100)::numeric,1) as d180
FROM cohort_roas_data
WHERE spend > 1000
GROUP BY campaign
ORDER BY d180 DESC;

-- Обновить когорты
cd ~/scripts/qonversion && node worker/update-cohorts.js
```
