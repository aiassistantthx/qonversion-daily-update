# Исследование: Динамика конверсий и прогнозирование COP

## Резюме

**Ключевой вывод:** Конверсии из триала продолжаются значительно дольше, чем 30 дней. Анализ 13,081 конверсий показывает, что к 30 дням приходит только ~83% конверсий, а для достижения 98% требуется около 6 месяцев.

**Практическое применение:** Это позволяет прогнозировать финальный COP (Cost of Payer) для свежих когорт, что критично для принятия решений о рекламных расходах.

---

## Методология

### Источник данных
- **База:** PostgreSQL на Coolify
- **Период:** январь 2025 — март 2026
- **События:** Trial Converted + Subscription Started (yearly products)
- **Метрика:** Разница между `event_date` и `install_date`

### Критерии включения
- Только конверсии в платные подписки
- Годовые продукты (`product_id LIKE '%yearly%'`)
- Когорты возрастом 60+ дней (для полноты данных)

## Применение: Прогноз COP

### Формула

```
Predicted COP = Actual Spend / (Actual Subs / Decay Factor)
```

Где `Decay Factor` — процент конверсий, ожидаемых к текущему возрасту когорты.

### Пример расчёта

**Когорта 3 марта 2026 (возраст 5 дней):**
- Spend: $1,200
- Текущих подписчиков: 46
- Decay factor на день 5: 0.629 (62.9%)
- Ожидаемых подписчиков: 46 / 0.629 = 73
- Текущий COP: $1,200 / 46 = **$26**
- Predicted COP: $1,200 / 73 = **$16**

## Риски и ограничения

1. **Сезонность** — данные за 14 месяцев могут не учитывать годовые циклы
2. **Изменение продукта** — изменения в триале/ценах влияют на паттерн
3. **Молодые когорты** — для когорт моложе 60 дней данные неполные
4. **Выбросы** — отдельные месяцы показывают отклонение до ±4%

---

## Рекомендации

1. **Для свежих когорт (0-7 дней):** Использовать predicted COP для оценки
2. **Для когорт 1-2 месяца:** Фактический COP близок к финальному (±10%)
3. **Для когорт 6+ месяцев:** Можно использовать фактический COP как финальный
4. **Пересчёт decay curve:** Рекомендуется раз в квартал

### Код decay curve

Файл: `api/routes/dashboard.js`

```javascript
const getDecayFactor = (days) => {
  if (days >= 365) return 1.0;
  if (days < 0) return COP_DECAY_CURVE[0];
  const keys = Object.keys(COP_DECAY_CURVE).map(Number).sort((a, b) => a - b);
  for (let i = 0; i < keys.length - 1; i++) {
    if (days >= keys[i] && days < keys[i + 1]) {
      const t = (days - keys[i]) / (keys[i + 1] - keys[i]);
      return COP_DECAY_CURVE[keys[i]] +
             t * (COP_DECAY_CURVE[keys[i + 1]] - COP_DECAY_CURVE[keys[i]]);
    }
  }
  return COP_DECAY_CURVE[keys[keys.length - 1]];
};
```
